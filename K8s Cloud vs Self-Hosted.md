# Self-Hosted Kubernetes Service
---

* VM provisioned, created and maintained by developers
* E.g. VirtualBox

# Cloud-Hosted Kubernetes Service
---

* Kubernetes-as-a-Service, VM provisioned, created and maintained by providers
* E.g. **GKE** (Google Kubernetes Engine), **AKS** (Azure Kubernetes Service), **EKS** (AWS Elastic Kubernetes Service)
